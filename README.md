##UPDATE FILP VER.5.0


보안 및 로직 개선 사항
​리뷰 및 닉네임 입력 시 발생할 수 있는 크로스 사이트 스크립팅 취약점을 방지하기 위해 이스케이프 함수를 추가하여 화면에 출력되도록 수정
​모달창이 나타나고 사라질 때 디스플레이 속성과 애니메이션 클래스 사이의 타이밍 문제를 해결하여 시각적 어색함을 개선
​리뷰 등록 버튼 클릭 시 서버 지연 시간 동안 다중 클릭이 발생하지 않도록 비활성화 및 텍스트 변경 처리 추가
​로컬 스토리지에 저장된 테마 값이 페이지 새로고침 시 즉각 반영되지 않던 로직 흐름 수정
​사용자 인터페이스 및 디자인 개선 사항
​기존 기본 폰트를 가독성과 현대적인 느낌이 우수한 프리텐다드 폰트로 전면 교체
​선으로 나뉘던 딱딱한 탭 메뉴를 둥근 캡슐형 세그먼트 컨트롤 디자인으로 변경하여 직관성과 조작감 향상
​모달창의 디자인을 화면 하단에서 부드럽게 올라오는 바텀 시트 형태로 개선하고 배경에 블러 효과 추가
​전체적인 카드 요소들의 테두리 반경을 넓히고 부드럽고 깊이감 있는 그림자를 적용하여 시각적 피로도 감소
​색상 대비를 고려해 텍스트와 배경 사이클의 여백과 간격을 넓혀 정보 구조를 명확하게 정돈
​알레르기 설정 안내 문구를 추가하고 경고 표시가 메뉴 텍스트와 자연스럽게 어우러지도록 레이아웃 재배치
​별점 및 상태를 나타내는 요소의 애니메이션 트랜지션 곡선을 수정하여 사용자 인터랙션 반응을 더 자연스럽게 조정
