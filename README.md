## FILP VER.1.0 HAS RELEASED!

### UI/UX 개선
- Bento UI 스타일 적용: 모듈화된 레이아웃으로 정보 구조 개선
- 글래스모피즘 효과 추가: 헤더와 달력 팝업에 반투명 블러 효과 적용
- 마이크로 인터랙션 강화: 버튼 호버 시 스케일 변환, 부드러운 전환 애니메이션
- 시스템 다크모드 자동 감지: OS 설정에 따라 자동 테마 전환
- 데스크톱 모달 중앙 정렬: 모바일은 바텀시트, 데스크톱은 중앙 모달로 분리
- 스켈레톤 로딩 개선: 더 세련된 그라데이션 애니메이션
- 알레르기 필터 접기/펼치기 기능 추가: 공간 효율성 개선
- 반응형 레이아웃 최적화: 모바일에서 주간 패널이 아래로 배치

### 신규 기능 추가
- 메뉴 검색 기능: 실시간 메뉴 필터링
- 즐겨찾기 기능: 마음에 드는 메뉴 하트 표시 및 localStorage 저장
- 리뷰 좋아요 기능: 다른 학우의 리뷰에 좋아요 추가/취소
- 리뷰 정렬 기능: 최신순, 별점 높은순/낮은순, 좋아요순 선택 가능
- 주간 인기 메뉴 TOP 3: 이번 주 가장 높은 평점을 받은 메뉴 표시
- 메뉴 공유 기능: Web Share API 또는 클립보드 복사 지원
- 키보드 네비게이션: ESC로 모달 닫기, 엔터/스페이스로 날짜 선택 및 별점 입력

### 성능 최적화
- Firebase 쿼리 결과에 ID 포함: 좋아요 업데이트 시 문서 참조 가능
- 이벤트 위임 최적화: 동적 요소에 대한 이벤트 처리 개선
- 불필요한 렌더링 방지: 검색 시 전체 메뉴 재파싱 없이 필터링만 수행

### 접근성 개선
- ARIA 레이블 추가: 모든 인터랙티브 요소에 적절한 aria-label 설정
- 키보드 포커스 표시: :focus-visible로 명확한 포커스 인디케이터 제공
- 역할(Role) 정의: 탭, 라디오 버튼, 다이얼로그 등에 적절한 ARIA 역할 부여
- reduced-motion 지원: 애니메이션 감소 설정 시 자동 비활성화
- 색상 대비 개선: 텍스트와 배경 간 가독성 향상

### 보안 강화
- HTML 이스케이프 함수 유지: XSS 방지를 위한 입력값 검증 계속 적용
- Firebase 보안 규칙 권장: 추후 서버사이드 검증 추가 권장
